📡 Core API Endpoints
POST/api/v1/audit/submit
Submit a lease PDF for full audit. Returns job_id. Supports multipart/form-data with jurisdiction, client_id, and lease_pdf fields.
GET/api/v1/audit/{job_id}/status
Poll audit progress. States: queued → ocr_running → extracting → auditing → human_review → complete.
GET/api/v1/audit/{job_id}/report
Download forensic report JSON or PDF. Includes all findings, clause references, and dispute amounts.
POST/api/v1/clients/onboard
Register a new client tenant under your partner account. Returns client_id for audit submissions.
GET/api/v1/clients/{client_id}/portfolio
Retrieve full audit history, monitoring status, and savings summary for a client.
